# Binance Funding

Python starter project for fetching and exploring Binance Futures funding rates.

## Quick Start

1. Create and activate a virtual environment.
2. Install the project in editable mode:

```bash
pip install -e .
```

3. Run the CLI:

```bash
binance-funding --symbol BTCUSDT --limit 5
```

## Development

Run tests:

```bash
pytest
```
